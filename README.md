### Transition and movimentation of widgets in Python Tkinter ###

Função: mover uma tela já colocada (com o método place) para as coordenadas dadas.
Caso o widget a ser movido tenha sido colocado (place) utilizando rel em qualquer uma das variáveis de posição, deverá ser dado em rel também para o método.

Function: move a widget from a locate to another by transition method. 
This function can operate with relative and specific method of position.

So much useful to develop a tkinter program.
